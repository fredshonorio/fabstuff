Additions to fabric

Get:

`pip install git+https://github.com/fredshonorio/fabstuff.git`

To install locally (dev): `pip install -e .`


Requires python2 (sorry), and fabric 1 (sorry).
In Archlinux and friends:

```
sudo pip2 install fabric==1.13.1
sudo pip2 install git+https://github.com/fredshonorio/fabstuff.git
```
